# ChainView — Supply Chain Mapper

A free, self-contained supply chain mapping tool for companies that 
don't have a dedicated supply chain function.

**No account. No server. No cost. One HTML file.**

[→ Launch the app](https://[your-username].github.io/chainview/)

## Features
- Visual flow graph with tier-based layout
- Geographic map view (city/state/country geocoding)
- Customisable tiers with names and colours
- Node-level operations with status flags (Active/Degraded/Offline)
- Link-to-operation assignment
- Spreadsheet import (.xlsx / .csv)
- AI analysis brief export for use with Claude, ChatGPT, or any AI tool
- Save/load as JSON, export as SVG or CSV
- Runs entirely in the browser — nothing is sent to any server

## How to use
Download `index.html` and open it in any modern browser, or use the 
hosted version linked above.
